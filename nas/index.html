Windows NAS新手向导教程和配置

发表于2025-10-11|更新于2026-05-29|监控安防
|浏览量:

  写在前面，写这个文章的目的，是备份这个操作，时间久了，可能会忘记，毕竟不是天天配置NAS。这个文章详细内容来自Bilibili的一个UP主，我只是简单的整理了一下，备用。

Windows NAS新手向教程 P1-硬件选择、系统安装、网络配置
一、硬件博主使用的四i3-12300t,核心8线程，2.3-4.2GHZ频率，35W功耗，带核显UHD730，这性能接近12100F接近，功耗不高性能很强。NAS神U。
低功耗移动端CPU:J4125/N5095/N5105/N100..
桌面端：I38100T/I312300T/I312100..
主板上E5+主板。
二、系统安装和配置：
使用专业版WIN10，安装完系统用单机账号即可。
1控制面板-系统和安全-更改用户账户控制设置-UAC通知关掉。
2.系统设置-网络和Internet-属性-把网络改成专用网络（如果没有属性的可以点菜单-运行-输入secpol.msc打开本地安全策略-左边打开网络列表管理策略-右边网络-网络位置-专用网络-权限-不可以更改位置）
3 网络和共享中心-高级共享设置-网络发现-启用，文件和打印机-启用

三、用户和共享
1修改设备名称，设置-关于-重命名
2 计算机管理-进入本地用户和组-组-新建组起个名族比如WINNAS
3 计算机管理-进入本地用户和组-用户-新建用户-取个用户名-打开密码永不过期。
4进入本地用户和组-组-右键新建的那个组-选择添加-选择用户，输入那个新建的用户名点检查名称。
四、共享文件夹
计算机管理-右边点共享-新建共享-文件夹路径-点浏览选择一个文件夹-权限-选择自定义权限-删除EVERRYONE-添加-新建的那个用户名（或者推荐添加组进去）-点检查名称-完全控制。

  详细内容来源哔哩哔哩：https://www.bilibili.com/video/BV1CCYCeuERr?spm_id_from=333.788.videopod.sections&vd_source=a58803799e4df6fa9aa8c8dc77c94a8d

东平白佛山：一场千年邂逅
  在城市的喧嚣中久了，便渴望寻一处宁静之地，让心灵得到片刻的栖息。听闻东平白佛山有着独特的魅力，在暑假的尾巴我终于按捺不住那份向往，踏上了这场探秘之旅。  当我站在白佛山脚下，抬眼望去，静静地矗立在天地之间。山上树木葱茏，在阳光的照耀下，闪烁着点点翠绿的光芒，仿佛是大自然为它披上的一件华美的衣裳。沿着蜿蜒的小径缓缓前行，脚下的石板路发出清脆的声响，仿佛在诉说着古老的故事。路边的野花野草肆意地生长着，它们在微风中轻轻摇曳，散发出淡淡的芬芳，让人仿佛置身于一个梦幻般的世界。  随着脚步的深入，在门右边的右边——石窟造像群。这些造像历经千年的风雨洗礼，依然栩栩如生，仿佛在向世人展示着古代工匠们的高超技艺和智慧。佛像的线条流畅自然，雕刻细腻入微，每一处细节都彰显着工匠们的匠心独运。  左面是漂流和花海，各种各样的美丽的鲜花在夏日阳光下随风摇曳，一株株各式各样的鲜花竞相争艳，还有订婚的新人在此举行仪式，这些花海见证这对对的缘分在此诞生，续写着一对对的人生篇章。  我继续向山顶攀登。越往上走，视野越开阔。当我终于站在山顶的那一刻，眼前的美景让我不禁为之惊叹。整个东平县城尽收眼底，那错落有致的建筑，那纵横交错的街道，那川流不息的人群，构成了一幅生机勃勃的画卷。远处的山峦连绵起伏，与蓝天白云相映成趣，仿佛是一幅天然的水墨画。微风吹过，带来了清新的空气和山间的鸟语花香，让人心旷神怡。  在山顶的一块巨石上，我静静地坐了下来，闭上眼睛，用心去感受着这一切。阳光洒在身上，暖暖的，仿佛是大自然给予我的拥抱。那一刻，我仿佛与这山、这水、这天地融为一体，所有的烦恼和疲惫都烟消云散。  不知不觉，夕阳的余晖洒在了白佛山的每一个角落。我知道，这场探秘之旅即将结束。但白佛山的美景、那千年的造像、那宁静的氛围，已经深深地印在了我的脑海里。我带着满满的收获和不舍，踏上了归途。  白佛山，你是大自然的杰作，是历史的见证者。你用你的宁静与神秘，让我在喧嚣的世界里找到了一片心灵的净土。我相信，在未来的日子里，我一定会再次踏上这片土地，与你再次邂逅，感受你那永恒的魅力。

第二次来大明湖
  第一次来大明湖好像是2015年，那个时候也是第一次去看趵突泉，大明湖在济南的政府正北面，湖不是很大，但是想走着转一圈还是花两个小时的，也只是走走停停看了西半边。第一次来的时候大明湖还是收费的，这次已经变成开放式的公园了，这样的惠民政策值得全国推广，  ‘四面荷花三面柳，一城山色半城湖’大明湖夏天去看的话真的很好看，湖中很清澈还有很多鱼，湖岸上也有很多历史遗迹等，拍照打卡好地方，游览湖光景色可以坐船，半小时好几十块。  大明湖景色确实宜人，从西南门出来正南放向就是趵突泉公园。  来看趵突泉的明显就比游大明湖的人多，每到节假日的时候那真是肩膀挨着肩膀，人是特别的多，今年去的时候大明湖已经免费开放了，趵突泉还是要40多块的，趵突泉的泉水真的很凉很清，天下第一泉，来这里一般都集中在公园中心这里。这里泉水不断，可以欣赏。来济南给我最大的感觉就是走到哪人都是特别的多，晚上去了芙蓉街，城中村的街道，街道很窄但是人特别多，这里主要就是小吃、美食为主，逛一圈下来卖旅游纪念的东西的还真不多。  这次来发现这条街上新开了很多家那种销售80年代的复古的食品和物品的店了，人越来越追忆了。离开济南的时候停车费一晚居然要80元，好家伙，真贵啊，记得上次来好像不收费啊。后来听朋友说可以找旅店老板报销的这个我是真不知道啊，下次就知道了。
